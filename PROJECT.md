# PROJECT.md

## プロジェクト名
CareerSelect（キャリアセレクト）

## プロジェクト目的
キャリアコンサルタント個人の実績と口コミを可視化し、求職者が“担当者を選べる”ようにするWebサービスを提供する。
ミドル〜ハイクラス層の転職を検討する求職者が、自分に合ったキャリアコンサルタントを選びやすくすることを目的とする。

## 機能一覧（MVP/POC範囲）
- 【求職者向け】
  - コンサルタント一覧表示
  - コンサルタント詳細・プロフィール・口コミの閲覧
  - 相談予約
  - 相談後の口コミ投稿（相談レビュー / 成果レビュー）
- 【コンサルタント向け】
  - ログイン
  - 自分のプロフィール編集
  - 相談予約の一覧確認
  - 相談ステータス（予定/完了）の変更
- 【運営/管理】
  - データ確認（GUIなしでもよい）

## 技術スタック
- Next.js（App Router）
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL（開発環境ではSQLiteも可）
- 最低限の認証（認証基盤は拡張可能な設計）
